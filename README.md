# Minecraft Arrears Edition

脑子一抽做的，用 Java + LWJGL 从头写的一个 Minecraft 风格原型。

说是游戏，其实更像一个技术验证。目前就是在一个平坦的世界里，用草方块和石头方块搭搭拆拆，支持第一人称移动、跳跃、重力、碰撞，左键拆右键放，按 ESC 可以保存退出。

代码一共七百行出头，从窗口创建、3D渲染、纹理加载、射线检测、AABB碰撞到GZIP压缩存档，全都手写。没有用任何游戏引擎，只依赖LWJGL这个底层图形库。

## 为什么做这个

不是想复刻Minecraft，纯粹是想搞清楚一件事：一个方块世界，从零到能走能跳能放能拆，到底需要写多少代码。答案是一千行以内。

做完发现，Notch在2009年做第一个版本时，核心逻辑和这个差不多。也算是一种致敬吧。

## 关于“简陋”这件事

运行画面确实很简陋。没有背包，没有合成，没有生物，没有多人，地面是一整块平整的平台。界面只有一行版本号和十字准星。

但它有碰撞检测，站在方块边缘不会掉下去，跳起来能踩到更高的方块上。有射线瞄准，十字指着的方块会被白框高亮。放方块的时候不会把自己卡进墙里。存档之后退出，下次进还是同一个地方。

这些最基础的东西，都是自己一行行调出来的。

## 怎么运行

你需要先安装Java运行环境，JDK 8到17应该都可以。下载仓库里的`Game.jar`和同目录下所有文件，保持它们在同一文件夹中，双击`Game.jar`启动。如果双击没有反应，可以使用`Launcher.exe`启动游戏。

## 控制方式

- **WASD** 移动
- **鼠标** 环顾四周
- **空格** 跳跃
- **鼠标左键** 拆除方块
- **鼠标右键** 放置方块
- **ESC** 保存并退出

按ESC会保存当前进度并退出游戏，下次启动会回到退出时的地方。

## 为什么没有把纹理一起打包

仓库里只写了加载纹理的代码，但没有上传`grass.png`和`stone.png`。如果你希望看到贴图效果，需要在运行目录下创建`data/Material`文件夹，自己放两张同名png进去。不做这一步也能跑，只是方块会变成纯色。

你可能注意到仓库里还有一堆`.dll`文件和`.exe`启动器。那些都是LWJGL在不同平台上的原生依赖，用来处理窗口、鼠标键盘输入和OpenGL加速。没有它们Java程序没法直接跟显卡和操作系统打交道。

## 最后

如果你只是想玩一个完整的游戏，这个项目大概率会让你失望。但如果你想看看一个方块世界的骨架可以怎么写，或者想改一改让它变成你自己的东西，这里的代码或许能给你一些参考。
